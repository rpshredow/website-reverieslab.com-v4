Welcome To My Website

I post my projects here and practice web development.

I'm rebuilding this website from scratch not using any frameworks like Django or Laravel, like I did on previous versions of this website or using any CSS libraries such as bootstrap. Although I'll probably use premade icons and fonts. Also, I won't be using any premade HTML/CSS templates.

Regular updates of this site will be easy as I have set up a CI/CD pipeline in AWS using Codepipeline. So I only have to push changes to the main branch of the repo in github and Codepipeline will automatically sense changes and deploy the changed files to a AWS S3 bucket set up for static website hosting.
